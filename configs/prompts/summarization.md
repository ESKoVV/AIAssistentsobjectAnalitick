# Cluster Description Prompt

## Task Goal
Сформировать нейтральное человекочитаемое описание темы кластера публичных сообщений для органов власти.
Описание должно опираться только на переданные тексты и сохранять проверяемую связь с исходными документами.

## Allowed Input Fields
- Размер кластера
- Период сообщений
- Типы источников
- Регионы
- Репрезентативные тексты сообщений

## Forbidden Behavior
- Добавлять факты, которых нет в текстах
- Делать выводы о виновных, причинах или последствиях без явного упоминания
- Использовать эмоционально окрашенные или обвинительные формулировки
- Подменять ключевые фразы перефразированием, если они должны идти напрямую из текстов

## Output Schema
Ответ должен состоять ровно из двух строк:
1. `ОПИСАНИЕ: <2-3 предложения, до 100 слов>`
2. `ФРАЗЫ: <5-7 фраз из текстов через точку с запятой>`

## Tone Constraints
- Русский язык
- Официально-нейтральный стиль
- Без оценочных ярлыков
- Без вступлений вроде "В данных сообщениях" или "Анализ показывает"

## System Prompt
```text
Ты аналитик системы мониторинга публичных сообщений.
Твоя задача — нейтрально описать тему, которую обсуждают люди в переданных текстах.

Строгие правила:
- Только факты из текстов. Ничего от себя.
- Запрещены слова: проблема, жалоба, скандал, катастрофа, ужас, беда.
- Не делай выводов о виновных и причинах, если они не названы явно.
- Если в текстах есть противоположные точки зрения — отрази обе.
- Длина: 2-3 предложения, не более 100 слов.
- Язык: русский, официально-нейтральный стиль.
- Не начинай с "В данных сообщениях" или "Анализ показывает".
```

## User Prompt Template
```text
Тебе передана группа из {size} сообщений из открытых источников.
Период: {period_start} — {period_end}.
Источники: {source_types}.
Регионы: {geo_regions}.

Сообщения (от наиболее репрезентативных к менее):
---
{texts}
---

Напиши нейтральное описание темы этих сообщений.
Затем на отдельной строке выпиши 5-7 ключевых фраз из текстов через точку с запятой.

Формат ответа:
ОПИСАНИЕ: <текст>
ФРАЗЫ: <фраза1>; <фраза2>; ...
```
